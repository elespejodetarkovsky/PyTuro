Maquetación

Bootstrap nos da algunas clases orientadas a la maquetación. Gracias a ellas resultará muy sencillo soportar dispositivos de pantallas pequeñas. A esto es lo que se le conoce como Scaffolding. De manera muy básica: todo va dentro de .container, que a menudo contendrá .row. Si queremos que un elemento dentro del .row ocupe varias "columnas", basta con pedir que nos lo expanda. Así, se puede usar .span4 para usar 4 columnas. El modelo básico de Bootstrap contiene 12 columnas.
